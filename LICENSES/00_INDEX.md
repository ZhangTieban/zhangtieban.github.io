## Notes

- This firmware build does not link against `Arduino_DriveBus`.
- FT3168 touch support is implemented in product-side source code.
- `Arduino-ESP32 Core` is treated as LGPL-2.1-or-later per upstream declaration.
- Generated UI fonts include glyphs derived from Source Han Sans SC.
- LVGL built-in text/icon fonts include Montserrat and Font Awesome Free.
- This folder must ship together with `THIRD_PARTY_NOTICES.txt`.
